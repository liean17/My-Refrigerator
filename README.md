# My-Refrigerator
냉장고 내 음식 관리 애플리케이션.  
[배포 주소](https://kdksandbox.shop)

### ERD
![refrigeratorERD](https://user-images.githubusercontent.com/94332594/199696995-e4814637-c6e1-4c61-8bf0-99b096d7f5d6.png)
### 애플리케이션 설명
- 냉장고에 음식을 등록할 수 있습니다.
- 냉장고는 음식 정보에 포함된 유통기한을 사용해서 유통기한을 관리해줍니다.

### 사용기술
 - Java 17
 - Spring Boot 5.3
 - Thymeleaf
 - Spring Data JPA
 - QueryDSL
 - MySQL 8.0.3
 - Spring Security
 - Spring Batch
 - AWS EC2
